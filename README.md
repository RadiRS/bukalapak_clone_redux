## Project Arkademy

### Bukalapak Clone

Project minggu ketiga bootcamp **Arkademy**. Membuat aplikasi E-Commerce dengan ketentuan **Product List, Product Detail, Cart, Payment Page, Register Page, Login Page, Profile Page**

### Tech :

- React Native
- Redux
- Redux Thunk
- Redux Promise Middleware
- React Navigation
- Native Base
- AdonisJS
